цикл for

  • в круглых скобках, после for происходит подготовка и частичное выполнение цикла
  • в фигурных скобках происходит изменение данных // или состояния системы
  • подготовка выполняется один раз: let i = 0; и затем эта запись игнорируется при всех последующих шагах цикла
  • выполнение происходит на каждом шаге цикла:
    • в начале выполнения шага: i < arr.length; // проверка происходит до входа в открывающую фигурную скобку {
    • в конце выполнения шага: i += 1 // изменение счетчика происходит при достижении закрывающей фигурной скобки }
  • наиболее распостранненые ошибки: неверно задан ключ окончания массива
